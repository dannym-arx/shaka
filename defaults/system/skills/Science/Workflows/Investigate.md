# Investigate Workflow

Full scientific method investigation for complex problems requiring systematic experimentation.

## Process

### 1. Observation Phase

Gather comprehensive data about the current state:

```markdown
## Observations

**Context**: [What is the situation?]

**Data Points**:

- [Quantitative observation 1]
- [Quantitative observation 2]
- [Qualitative observation 1]

**Patterns Noticed**:

- [Pattern 1]
- [Pattern 2]

**Anomalies**:

- [Unexpected finding 1]
```

### 2. Hypothesis Formation

Generate multiple testable hypotheses (minimum 3):

```markdown
## Hypotheses

| #   | Hypothesis  | If-Then Prediction   | Falsifiable?              |
| --- | ----------- | -------------------- | ------------------------- |
| H1  | [Statement] | If H1 true, then [X] | Yes - can disprove by [Y] |
| H2  | [Statement] | If H2 true, then [X] | Yes - can disprove by [Y] |
| H3  | [Statement] | If H3 true, then [X] | Yes - can disprove by [Y] |

**Priority Order**: H[n] > H[m] > H[p] because [reasoning]
```

### 3. Experiment Design

Design controlled experiments for each hypothesis:

```markdown
## Experiment: Testing H[n]

**Independent Variable**: [What you're changing]
**Dependent Variable**: [What you're measuring]
**Control**: [What stays constant]

**Procedure**:

1. [Step 1]
2. [Step 2]
3. [Step 3]

**Success Criteria**: [How you'll know if hypothesis is supported]
**Sample Size**: [If applicable]
```

### 4. Execution

Run experiments systematically:

- Execute in priority order
- Document all observations
- Note any deviations from procedure
- Record unexpected findings

### 5. Analysis

Analyze results objectively:

```markdown
## Results

| Hypothesis | Predicted | Observed | Delta | Supported?     |
| ---------- | --------- | -------- | ----- | -------------- |
| H1         | [X]       | [Y]      | [Δ]   | Yes/No/Partial |
| H2         | [X]       | [Y]      | [Δ]   | Yes/No/Partial |
| H3         | [X]       | [Y]      | [Δ]   | Yes/No/Partial |

**Statistical Confidence**: [If applicable]
**Potential Confounds**: [What might have affected results]
```

### 6. Conclusion & Iteration

```markdown
## Conclusion

**Primary Finding**: [What we learned]
**Confidence Level**: High/Medium/Low
**Limitations**: [What this doesn't tell us]

**Next Steps**:

- [ ] [Follow-up experiment 1]
- [ ] [Action based on findings]
- [ ] [New hypothesis to test]
```

## Output

Deliver complete scientific report with all sections above.
